“regrrr”: One-stop R-toolkit for Compiling Regression Results

2019/03/06

Today, I released an R package “regrrr” on Cran. In strategy/management research, we always need to compile the regression results into the publishable format and sometimes plot the moderating effects. This package does the job.

library(regrrr)
data(mtcars)
m0 <- lm(mpg ~ vs + carb + hp + wt, data = mtcars)
m1 <- update(m0, . ~ . + wt * hp)
m2 <- update(m1, . ~ . + wt * vs)
cor.table(data = m2$model)

As you can see from the last line of code, the plot is customizable using “ggplot2”. There are a couple of other function. Please see the reference manual for details.

I’m also aiming to expand the package’s usage around its core functions. If you have any ideas or want to report a bug, please contact me or suggest on the GitHub page.
